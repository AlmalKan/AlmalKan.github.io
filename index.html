    About

    Nuestra casa domótica es un innovador sistema que integra tecnologías de automatización para transformar el hogar en un espacio inteligente y eficiente. Mediante el uso de dispositivos como Arduino, módulos Bluetooth, LEDs, servomotores y pantallas LCD, hemos creado un entorno interactivo y personalizable que se adapta a las necesidades y preferencias individuales.

    Arduino

    Arduino es una plataforma de prototipado electrónico de código abierto. Consiste en una placa con un microcontrolador que puede ser programado para realizar diversas funciones. Es muy utilizado para proyectos de electrónica y robótica.

    Modulo Bluetooth

    El módulo Bluetooth permite la comunicación inalámbrica entre dispositivos. Puedes conectarlo a Arduino para habilitar la comunicación con otros dispositivos como teléfonos móviles o tablets a través de la tecnología Bluetooth.

    Protoboard

    Una protoboard (o placa de pruebas) es una herramienta que permite conectar componentes electrónicos sin la necesidad de soldar. Es útil para prototipos y proyectos temporales, ya que los componentes pueden colocarse y retirarse fácilmente.

    Leds

    Los LEDs son dispositivos electrónicos que emiten luz cuando se les aplica corriente. Son comúnmente utilizados para indicar estados, como encendido o apagado. También pueden ser parte de proyectos de iluminación y decoración.

    Servormotor

    Un servomotor es un tipo de motor que se utiliza para controlar el movimiento de objetos. Puede girar hasta cierto ángulo según las señales que recibe. Es comúnmente utilizado en proyectos donde se requiere control preciso del movimiento, como abrir una puerta o mover un brazo robótico.

    Pantalla Lcd

    Las pantallas LCD son dispositivos de visualización que utilizan cristales líquidos para producir imágenes. Pueden mostrar texto, gráficos y otros datos. En proyectos de Arduino, las pantallas LCD son utilizadas para mostrar información como sensores, mensajes o estados del sistema.
